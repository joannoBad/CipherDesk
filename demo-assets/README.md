# Demo Assets

Положи сюда файлы, на которых нужно автоматически переснимать скриншоты для `README.md`.

Рекомендуемая структура:

- `text-encrypt-input.txt` — исходный текст для режима шифрования текста
- `text-decrypt-input.json` — создаётся автоматически из исходного текста
- `image-input.jpg` / `image-input.png` — исходная картинка
- `image-encrypted.cdesk` — создаётся автоматически для сценария decrypt
- `document-input.pdf` — исходный документ
- `document-encrypted.cdesk` — создаётся автоматически для сценария decrypt
- `decrypt-error.json` — создаётся автоматически как плохой payload для скрина ошибки
- `demo-manifest.json` — создаётся автоматически и хранит demo-пароль

Если имена будут другими, их можно будет прописать в `screenshot-scenarios.json`.
